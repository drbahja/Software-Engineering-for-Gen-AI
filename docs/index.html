🎯 What This Course Is About

Generative AI models like DeepSeek can write, reason, and answer questions — but turning a model into a real application requires software engineering: clean project structure, secure handling of API keys, version control, testing, and deployment. That is what you will practise here.

By the end of the 4 days you will be able to:

  • Explain how large language models work — enough to make good engineering decisions
  • Build AI agents with LangChain and the DeepSeek API
  • Structure a professional Python project: requirements.txt, .env, .gitignore, Git & GitHub
  • Build retrieval (RAG) and tool-using agents that work with your own data
  • Evaluate, debug, and deploy a working Gen AI application as a final project

Format: each day starts in the lab — you code first, then we discuss the ideas behind what you built.
